## Overview

Built a serverless video analytics architecture that processes uploaded surveillance footage and generates real-time alerts.

### Key Features
- **Face & Object Detection** using AWS Rekognition
- **Event-driven processing** with S3 triggers and Lambda
- **Real-time notifications** via SNS
- **Fully serverless** - no infrastructure to manage
