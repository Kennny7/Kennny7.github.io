## Overview

Built a cross-platform desktop productivity tool that enforces focus by detecting window activity and penalizing distractions in real time.

### Key Features
- **Focus Monitoring** detects minimize events and window switching
- **Punishment System** triggers sound, alerts, and optional timer reset
- **Pomodoro Timer** configurable focus and break cycles
- **Local Persistence** stores sessions and stats without external services
- **System Tray Support** enables background execution and quick restore
