## Overview

Built a mobile-first MCQ scanning system that extracts questions from images and returns probable answers using OCR and search-based inference.

### Key Features
- **Real-time image capture** using Flutter camera integration
- **OCR-based text extraction** via OCR.space API
- **Automated answer retrieval** using search result parsing
- **API-driven architecture** powered by FastAPI backend
- **Cross-platform mobile app** with responsive Material UI
