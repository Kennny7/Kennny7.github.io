## Overview

End-to-end streaming analytics pipeline on Azure that ingests live crime data, processes in real-time, and serves predictions via a REST API.

### Key Features
- **Stream ingestion** via Azure Event Hubs
- **Real-time processing** with Stream Analytics
- **ML model deployment** as REST endpoint
- **Live dashboards** in Power BI
