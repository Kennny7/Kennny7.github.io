## Overview

Built a modular AI orchestration engine that enables autonomous assistants to plan, execute, and iterate on real-world tasks.

### Key Features
- **Agentic execution loop** with planning, action, and feedback iteration
- **Plugin-based architecture** for OS, web, memory, and speech integration
- **Real-time streaming** via WebSockets for interactive responses
- **Vector-based memory** for long-term context and recall
- **Cross-platform client** using Flutter with voice and chat interface
